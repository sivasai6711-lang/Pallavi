# Event Registration System

This is a simple Event Registration System web application.
Users can register for events through a frontend interface, and the backend server stores and processes the registration details.

## Project Structure

Backend

* server.js
* package.json

Frontend

* index.html
* style.css
* script.js

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js

## Features

* User event registration
* Backend server handling requests
* Simple and responsive frontend
* Data submission from frontend to backend

## How to Run the Project

### 1. Install dependencies

Navigate to the backend folder and install required packages:

npm install

### 2. Start the server

Run the following command:

node server.js

### 3. Open the frontend

Open the index.html file in your browser.
